Netflix Clone
Esse projeto foi feito durante a imersão de front-end com IA da Alura.
A proposta foi criar uma interface inspirada na Netflix, com seleção de perfis e navegação por um catálogo estilo streaming. 
Parte do catálogo foi disponibilizada durante a imersão e utilizada no projeto.
Como ainda estou começando no front-end, usei inteligência artificial para gerar o código base. 
Depois disso, fiz ajustes simples por conta própria, como cores, fontes e pequenos detalhes no layout.
O projeto permite selecionar um perfil, navegar até o catálogo e mantém o perfil escolhido usando localStorage.
Também possui troca de tema claro e escuro como funcionalidade extra.
Usei a IA como apoio, mas buscando entender o que estava sendo feito. 
Durante o processo percebi que ela ajuda bastante, mas também pode errar ou complicar coisas simples, então fui testando, ajustando e aprendendo com os erros.
Mesmo sendo um projeto simples, isso me deu mais vontade de continuar aprendendo front-end.
Para rodar o projeto, basta abrir o arquivo index.html no navegador, escolher um perfil e navegar pelo catálogo
